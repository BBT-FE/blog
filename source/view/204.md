1. `js变量提升`：就是把变量提升提到函数的top的地方,变量提升 只是提升变量的声明，并不会把赋值也提升上来。
2. `函数提升`是把整个函数都提到前面去.只有函数声明形式才能被提升。
```js
  foo(); 
  function foo(){  // 函数声明形式
    alert("我来自 foo"); 
  } 

  var foo =function foo(){ } // 函数表达式形式
```
3. `匿名函数`:首先声明一个函数对象，然后执行它。
```js
  //几种形式
 1.(function () { alert(1);})();
 2.var a = function(){ return 1 }; // 函数function 没有函数名，但是它将值赋给了变量 a
```
